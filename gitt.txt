Git
	Git สร้างโดย Linus Torvalds เมื่อปี 2005 จุดประสงค์แรกๆ เพื่อใช้สำหรับการควบคุมเวอร์ชั่นของ Linux kernel เนื่องจาก Linux kernel มีโค๊ดจำนวนมากหลายล้านบรรทัด และ Linux kernel มีคนสร้างหลายคน ทำให้ version control system เดิม ไม่สามารถใช้กับ Linux kernel ได้อีกต่อไป จึงใช้ Git แทน ดูเพิ่มเติม
	หลังจากนั้น Git ก็ได้รับความนิยมอย่างแพร่หลายในหมู่ software developer, programmer และยังได้รับความนิยมอย่างสูงสุดจนมาถึงปัจจุบัน
	Git เป็นเครื่องมือ เพราะฉนั้นเราต้องเรียนรู้ที่จะใช้มัน ซึ่งรับรองเลยว่าเมื่อใช้ Git เป็นแล้วมันจะคุ้มค่ามากๆ จนลืมไปเลยว่าตอนเราเรียนรู้ที่จะใช้มันยากแค่ไหน
ใครใช้ Git บ้าง
	Git ใช้ได้ดีตั้งแต่โปรเจคส่วนตัวขนาดเล็ก ไปจนถึงโปรเจคระดับโลก ไม่น่าแปลกใจเลยที่บริษัทขนาดใหญ่ๆจะมาใช้ Git เช่น Google, Microsoft, Apple, True, Facebook และบริษัทอื่นๆที่เกี่ยวข้องกับซอฟท์แวร์อีกหลายร้อยหลายพันบริษัท
	ถ้าเราใช้ Git ได้ถือว่าเรามี skill ที่หลายๆบริษัทต้องการ
